TABLE

Ringkasan Materi

Tabel tersusun atas baris dan kolom, di mana setiap pertemuan baris dan kolom disebut sel. Struktur dasar tabel terdiri dari tag <table> sebagai pembungkus utama, <tr> untuk baris, <td> untuk sel data, dan <th> untuk header tabel. Beberapa atribut penting pada tabel antara lain border untuk menampilkan garis tabel, cellpadding untuk mengatur jarak isi sel dengan border, serta cellspacing untuk mengatur jarak antar sel. Selain itu, tabel dapat diberi judul menggunakan tag <caption> yang secara default ditampilkan di bagian atas tengah tabel.

HTML juga menyediakan atribut pengaturan posisi dan penggabungan sel untuk membuat tabel lebih fleksibel. Atribut align dan valign digunakan untuk mengatur perataan horizontal dan vertikal isi tabel, sedangkan colspan dan rowspan berfungsi untuk menggabungkan beberapa kolom atau baris. Dengan kombinasi atribut tersebut, tabel dapat digunakan tidak hanya untuk menampilkan data, tetapi juga untuk mengatur layout halaman web secara sederhana. Contoh latihan pada materi ini menunjukkan penerapan penggabungan baris dan kolom, penyisipan gambar ke dalam sel tabel, serta pengaturan tabel agar tampil rapi dan terstruktur di browser.
