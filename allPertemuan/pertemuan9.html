FRAME

Ringkasan Materi

Frame dalam HTML digunakan untuk membagi tampilan layar browser menjadi beberapa bagian berdasarkan baris (rows) dan kolom (cols). Dengan menggunakan elemen <frameset> dan <frame>, satu halaman web dapat menampilkan beberapa dokumen HTML secara bersamaan dalam satu layar. Setiap frame dapat memuat file HTML yang berbeda, sehingga memudahkan pengaturan tata letak seperti header, menu, konten utama, dan footer. Pada halaman yang menggunakan frame, tag <body> tidak digunakan karena sudah digantikan oleh <frameset> sebagai pengatur utama tampilan.

Elemen <frameset> memiliki atribut rows dan cols yang berfungsi untuk menentukan pembagian ukuran frame, baik dalam satuan piksel, persentase, maupun ukuran relatif. Frame juga dapat dikombinasikan secara bertingkat untuk membentuk layout yang lebih kompleks. Selain itu, frame sering dipadukan dengan hyperlink menggunakan atribut target agar halaman yang diklik dapat ditampilkan pada frame tertentu. Meskipun frame kini jarang digunakan pada pengembangan web modern, pemahaman konsep frame tetap penting sebagai dasar pengaturan layout dan navigasi pada HTML generasi awal.

Contoh Penerapan Frame

<html>
<head>
--- bagian head ---
</head>
<frameset>
--- bagian frame ---
</frameset>
</html>
