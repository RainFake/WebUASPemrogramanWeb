IFRAME

Ringkasan Materi

Iframe (Inline Frame) merupakan elemen HTML yang digunakan untuk menampilkan atau menyematkan dokumen lain ke dalam sebuah halaman web tanpa harus berpindah halaman. Dengan iframe, sebuah halaman dapat memuat konten eksternal seperti halaman web lain, video, peta, atau media interaktif langsung di dalam halaman yang sedang aktif. Tag iframe ditulis dengan pasangan <iframe> dan </iframe>, dengan atribut utama src yang berfungsi menentukan alamat dokumen atau konten yang akan ditampilkan. Jika browser tidak mendukung iframe, maka teks atau elemen di dalam tag iframe akan ditampilkan sebagai alternatif.

Iframe didukung oleh hampir semua browser modern seperti Google Chrome, Mozilla Firefox, Safari, Opera, dan Internet Explorer. Beberapa atribut iframe yang sering digunakan antara lain width dan height untuk mengatur ukuran tampilan, frameborder untuk menentukan ada atau tidaknya garis tepi, serta name yang memungkinkan iframe menjadi target dari hyperlink. Pada HTML5, sebagian atribut lama seperti align, scrolling, dan frameborder sudah tidak direkomendasikan, sehingga pengaturan tampilan iframe lebih dianjurkan menggunakan CSS. Dengan CSS, iframe dapat diatur secara fleksibel baik dari sisi ukuran maupun tata letaknya di dalam halaman web.

Perbedaan utama antara FRAME dan IFRAME terletak pada cara penggunaannya. FRAME membagi seluruh halaman browser menjadi beberapa bagian dan menggunakan <frameset> tanpa <body>, sedangkan IFRAME digunakan di dalam <body> dan hanya menyisipkan konten tambahan di suatu bagian halaman. Karena lebih fleksibel dan sesuai standar HTML modern, iframe lebih sering digunakan dalam pengembangan web saat ini dibandingkan frame.

Contoh Penerapan Frame

<HTML>
<HEAD>
<TITLE> New Document </TITLE>
</HEAD>
<BODY>
<h1>Penggunaan IFRAME | https://indarnirmala.wordpress.com/</h1>
<iframe src="https://indarnirmala.wordpress.com" height="550px" width="900px"></iframe>
</BODY>
</HTML>
