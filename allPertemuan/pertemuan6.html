BULLET NUMBERING DAN LIST

Ringkasan Materi

Bullet, Numbering, dan List dalam HTML membahas cara menampilkan daftar data secara terstruktur pada halaman web. Daftar ini sangat penting untuk menyajikan informasi secara rapi, mudah dibaca, dan sistematis. HTML menyediakan beberapa jenis list, di antaranya ordered list (numbering) untuk daftar berurutan, unordered list (bullet) untuk daftar tanpa urutan, menu list, serta definition list untuk menampilkan istilah beserta penjelasannya. Seluruh jenis list ini menggunakan tag khusus seperti <ol>, <ul>, <menu>, dan <dl> yang dipadukan dengan tag <li>, <dt>, dan <dd>.

Ordered list (<ol>) digunakan ketika urutan data bersifat penting, misalnya langkah-langkah atau peringkat, dan dapat ditampilkan dengan berbagai format penomoran seperti angka, huruf, atau romawi. Sementara itu, unordered list (<ul>) digunakan untuk menampilkan daftar tanpa urutan tertentu dan biasanya menggunakan simbol bullet seperti lingkaran atau kotak. Menu list berfungsi menampilkan daftar sederhana tanpa penomoran maupun bullet, sedangkan definition list (<dl>) digunakan untuk menjelaskan suatu istilah dengan deskripsi yang lebih rinci. Dengan memahami dan mempraktikkan penggunaan berbagai jenis list ini, mahasiswa dapat membuat tampilan halaman web yang lebih informatif, terstruktur, dan profesional sesuai kebutuhan pengembangan web dasar.

Contoh penerapan Jenis Numbering

Kota Besar di Jawa :
<ol type="A">   
<li>Jakarta</li>
  <li>Bandung</li>
  <li>Surabaya</li>
</ol>

Hasilnya :

Kota Besar di Jawa :
  1. Jakarta
  2. Bandung
  3. Surabaya
