BASIC TAG HTML

Ringkasan Materi

HTML (HyperText Markup Language) merupakan bahasa penanda yang digunakan untuk menyusun dokumen web. HTML menggunakan tag yang ditulis di antara tanda kurang dari dan lebih dari. Tag HTML umumnya bersifat berpasangan, terdiri dari tag pembuka dan penutup, meskipun terdapat pula tag tunggal serta tag yang dilengkapi atribut untuk mengatur perilaku dan tampilan elemen.

Struktur dokumen HTML tersusun atas dua bagian utama, yaitu bagian head dan body. Bagian head berisi informasi yang tidak ditampilkan langsung di halaman browser, seperti judul halaman, metadata, serta pengaturan dokumen. Sementara itu, bagian body berisi seluruh konten yang dapat dilihat oleh pengguna, seperti teks, gambar, tautan, dan elemen multimedia.

Elemen head memiliki peran penting dalam pengelolaan halaman web, khususnya untuk kebutuhan mesin pencari dan pengaturan tampilan. Di dalamnya dapat terdapat tag title untuk menampilkan judul halaman, meta tag untuk memberikan deskripsi dan kata kunci, serta tag lain yang berfungsi menghubungkan dokumen HTML dengan sumber eksternal. Penggunaan meta tag yang tepat dapat membantu mesin pencari memahami isi halaman web secara lebih baik.

Bagian body digunakan untuk menampilkan informasi yang akan dipublikasikan ke pengguna. Tampilan halaman web dibentuk dari kombinasi struktur HTML, pengaturan gaya menggunakan CSS, serta interaksi menggunakan JavaScript. Untuk mengatur format dokumen, HTML menyediakan berbagai tag seperti paragraf, heading, garis horizontal, teks terformat, dan indentasi, yang berfungsi memperjelas struktur dan meningkatkan keterbacaan konten.

Contoh Penggunaan Tag

<html>
<head>
  <title>Pengenalan Tag HTML</title>
  Berfungsi untuk menampilkan judul halaman pada tab browser
</head>

<body>
  Bagian body berisi seluruh konten yang ditampilkan di halaman web
  seperti teks, gambar, link, dan elemen lainnya.
</body>
</html>
