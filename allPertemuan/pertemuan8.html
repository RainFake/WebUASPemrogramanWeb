FORM

Ringkasan Materi

Form dalam HTML digunakan untuk mengumpulkan data dari pengguna dan mengirimkannya ke server untuk diproses lebih lanjut. Form sering dimanfaatkan dalam berbagai kebutuhan seperti pengisian biodata, pendaftaran online, umpan balik pengguna, hingga transaksi pembelian. Pembuatan form diawali dengan tag <form> yang memiliki atribut action sebagai tujuan pengiriman data dan method sebagai metode pengiriman data, yaitu GET atau POST. Metode GET biasanya digunakan untuk pengiriman data sederhana, sedangkan POST digunakan untuk data yang lebih kompleks dan bersifat sensitif.

Di dalam form terdapat berbagai jenis input yang berfungsi untuk menerima data sesuai kebutuhan. Input teks (text) digunakan untuk memasukkan angka atau tulisan, radio button (radio) digunakan untuk memilih satu opsi dari beberapa pilihan, sedangkan checkbox (checkbox) memungkinkan pengguna memilih lebih dari satu opsi. Selain itu, tersedia dropdown atau list (select) untuk menampilkan pilihan dalam bentuk daftar, serta textarea (textarea) untuk memasukkan teks dalam jumlah banyak atau lebih dari satu baris. Untuk mengakhiri proses pengisian form, digunakan tombol submit yang mengirimkan data dan tombol reset untuk mengosongkan kembali isian form.

Dengan memahami penggunaan form dan berbagai jenis input di dalamnya, pengguna dapat membuat halaman web yang interaktif dan mampu berkomunikasi dengan pengguna secara langsung. Form menjadi komponen penting dalam pengembangan website dinamis karena berperan sebagai penghubung antara pengguna dan sistem yang mengolah data di sisi server.

Contoh Form Table

<form action="url" method="get | post">
...........
...........
</form>
