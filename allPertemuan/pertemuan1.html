PENGENALAN WEB

Ringkasan Materi

World Wide Web (WWW) adalah kumpulan halaman web di seluruh dunia yang saling terhubung melalui hyperlink, memungkinkan pengguna mengakses informasi dengan cepat dan mudah tanpa harus membaca secara berurutan. Halaman web dibuat menggunakan HTML (Hypertext Markup Language), yaitu bahasa markup yang mengatur struktur dan tampilan konten web seperti teks, gambar, video, dan tautan. HTML bekerja bersama browser untuk menampilkan informasi secara visual kepada pengguna. Komunikasi antara browser dan server menggunakan HTTP, protokol yang bertugas mengirim data dari server ke perangkat pengguna. Sistem web bekerja dengan model client–server, di mana server menyimpan data dan browser menampilkannya. Web server seperti Apache berfungsi menyimpan, mengamankan, dan melayani halaman web serta mendukung pembuatan web dinamis menggunakan bahasa server-side seperti PHP. Struktur dasar HTML terdiri dari dua bagian utama:
Head → informasi halaman
Body → konten yang ditampilkan
Dengan teknologi web, informasi dapat diakses kapan saja, di mana saja, dan dalam berbagai bentuk multimedia.


Contoh Struktur HTML

<html>
  <head>
--- bagian head ---
</head>
  <body>
--- bagian body ---
</body>
</html>
